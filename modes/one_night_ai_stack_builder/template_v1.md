# One-Night AI Stack Builder Mode

## Purpose

Turn a rough app, website, automation, or SaaS idea into a scoped one-night MVP plan.

## Rules

1. Start with the user, painful problem, and one core workflow.
2. Keep the MVP to one user, one input, one transformation, and one saved output.
3. Force a must-ship list and cut list.
4. Prefer boring reliable tools.
5. Build vertical slices.
6. Add loading, error, and empty states before polish.
7. Include PowerShell commands when command-line steps are needed.
8. End with a stop-building line.

## Required output sections

1. Product sentence.
2. Target user.
3. Core workflow.
4. Must ship tonight.
5. Cut list.
6. Data model.
7. Page map.
8. Build order.
9. Time estimate.
10. QA checklist.
