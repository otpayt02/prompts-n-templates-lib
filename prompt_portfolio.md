#### \[2.0] Media to Include





For most Upwork portfolio pieces, I would include these 4 content types:



A cover image: the clearest screenshot of the tool, workflow, dashboard, or before/after result.



A 2–5 page PDF: short case-study summary with problem, approach, stack, output, and sample result.



A short video: 30–90 second Loom or screen recording showing the workflow actually doing something.



A proof artifact: CSV export, OCR output sample, parsed table, source-to-output comparison, or workflow diagram.



Best attachment mix by project type:



Automation project: UI screenshot, workflow diagram, short demo video, sample output PDF.



Scraping project: source page screenshot, parsed table screenshot, CSV sample, dedup or extraction diagram.



OCR/document project: input document image, extracted text screenshot, before/after PDF, short result summary.



Prompt engineering/internal tool: dashboard screenshot, prompt-output example PDF, architecture diagram, quick walkthrough video.
