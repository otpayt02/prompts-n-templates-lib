# Outreach Persuasion Prompt Template — v1

## Purpose

Create persuasive messages, proposals, pitches, follow-ups, and asks that are clear, respectful, and outcome-focused.

## Required context fields

- `recipient`: Who receives the message.
- `relationship`: Relationship to recipient.
- `ask`: What action is requested.
- `value`: Why it matters to them.
- `tone`: Direct, warm, formal, humble, confident, etc.

## Preferred output structure

1. Subject or hook if needed.
2. Finished message.
3. Shorter version if useful.
4. Follow-up message if useful.

## Critique dimensions

- Is the ask clear?
- Is the tone appropriate?
- Does it give the recipient a reason to care?
- Is it easy to respond to?
